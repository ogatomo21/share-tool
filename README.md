# 説明
アプリ内にあるHTMLコードを自サイトに貼りつけるだけで様々サービスに対応した共有ボタンを<span style="font-size: 130%;">一つで</span>作ることが出来ます。

# 使い方
[アプリ](https://share-tool.vercel.app)を開くと共有画面が出てきます。
フッターにある「埋め込みコードはこちら」をクリックして自身のサイトに貼りつけるだけでご利用できます。
